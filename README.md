# 🤿 Freediving Safety Buddy Simulator

다이버와 세이프티 버디의 미팅 타이밍을 시뮬레이션하는 웹 앱입니다.

## 🌐 바로 사용하기

**[👉 앱 실행하기](https://ftoaster.github.io/freediving-safety-sim/)**

## 📱 PWA 설치 방법

| 기기 | 설치 방법 |
|------|----------|
| **PC Chrome** | 주소창 오른쪽 설치 아이콘 클릭 |
| **Android** | Chrome → ⋮ → "홈 화면에 추가" |
| **iPhone/iPad** | Safari → 공유 → "홈 화면에 추가" |

## ✨ 기능

- 🎯 목표 수심 & 미팅 수심 설정
- ⏱️ 세이프티 출발 타이밍 자동 계산
- 🎬 타임라인으로 시뮬레이션 스킵/점프
- 📊 실시간 시각화
- 📱 PWA 지원 (오프라인 사용 가능)
